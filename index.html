About

My name is Jalaluddin Qureshi (Jalal). I am a full-stack web developer graduating out of Humber College Graduate Web Development program. As it can be seen from my Github repositories, I aim to write clean code, which are modular, commented, indented and well-documented. I enjoy solving problems, and have worked on projects in which I designed efficient algorithms for computational problems, during which I gained hands-on knowledge about data structure, and writing code which executes with minimal steps.

I have designed websites which are responsive and accessible, to cater the needs of wide variety of different users, as well as pay attention to the semantic meanings of HTML elements, and its attributes so that the site performs well when evaluated by a search engine optimization (SEO) evaluator.

Some of the representative projects I have completed on usability testing and web analytics include, cognitive walk through of a website, creating a dummy Shopify website and analyzing the characteristics of users visiting the website using Google analytics, and customer segmentation using PRIZM5 tool to improve online marketing. A demo of the Google analytics analysis of a Shopify website is given on this Youtube link.
